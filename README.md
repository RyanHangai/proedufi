# Proedufi

**Pro**jeto **Edu**cação **Fi**nanceira (Financial Education Project) aims to improve and encourage financial education, promoting healthier financial habits and offering accessible and easy-to-understand expense control.

# Installation

## Linux

## Build from source:

Clone and build from source

### Dependences:
- make
- gcc

* Remind: Before run the command to install, enter in your aplication paste. E.g: 
```
cd ~/Aplications
```

For clone and build, run the command bellow:

```
git clone https://github.com/RyanHangai/proedufi.git && cd proedufi && make
```

For execute run
```
proedufi
```
Remind: The software will only run if it is located in the folder where it was built.
