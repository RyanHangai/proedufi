# Proedufi

**Pro**jeto **Edu**cação **Fi**nanceira (Financial Education Project) is a project whose improve and incentivate the financial education for all people.

# Installation

## Linux

### Any distros:

Clone and build from source

* Remind: Before run the command to install, enter in your aplication paste. E.g: 
```
cd ~/Aplications
```

After enter in your paste, run the command bellow

```
git clone https://github.com/RyanHangai/proedufi.git && cd proedufi && make
```
