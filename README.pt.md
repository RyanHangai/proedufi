**Proedufi** (**Pro**jeto **Edu**cação **Fi**nanceira) é uma aplicação de linha de comando (CLI) escrita em **Rust**. Seu objetivo é melhorar e incentivar a educação financeira, promovendo hábitos financeiros mais saudáveis e oferecendo controle de despesas acessível e fácil de entender diretamente do seu terminal.

## Funcionalidades Planejadas
- **Gerenciamento de Usuários:** Criar contas, fazer login com segurança ou continuar como convidado.
- **Controle de Conta Familiar:** Gerenciar e monitorar múltiplas contas financeiras para seus familiares em um lugar centralizado.
- **Registro de Receitas e Despesas:** Controle abrangente do seu fluxo de caixa. Registre e acompanhe facilmente seus ganhos e gastos para manter um orçamento saudável.
- **Validação de Dados:** Sistema robusto para garantir nomes válidos, emails formatados corretamente e senhas fortes (exigindo maiúsculas, minúsculas, números e caracteres especiais).

## Pré-requisitos

Antes de começar, certifique-se de ter as seguintes dependências instaladas em seu ambiente Linux:

- `cargo`
- `rustc`

## Instalação e Configuração

1. **Navegue até o diretório de aplicativos desejado:**
   ```bash
   cd ~/Applications
   ```

2. **Clone o repositório e compile a partir do código-fonte:**
   ```bash
   git clone https://github.com/RyanHangai/proedufi.git
   cd proedufi
   make
   ```

## Uso

Para executar a aplicação, execute o binário gerado a partir da pasta do projeto:

```bash
./proedufi
```

> **Nota:** O software só funcionará corretamente se executado a partir da pasta onde foi compilado.
